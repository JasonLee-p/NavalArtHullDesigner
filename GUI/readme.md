# GUI模块

## 一.模块功能
### 通用控件：
- 存放图标、图片等资源文件
- 封装了 Qt 基础界面控件，例如按钮、标签、文本框等
- 实现了一些高级控件：例如拾色器，自定义对话框等
- 实现了一些更复杂的界面，例如多方向拖动的窗口，带有标题栏的窗口等
### 项目控件：
- 实现了项目的特定界面：例如主界面、工程文件组件编辑界面等

## 二.使用方法
### 1.通用控件
- 从 general_widgets 导入使用即可，具体方法参见 docstring；
- 若想扩展通用控件，请注意参考已有的控件封装，保持风格一致；

### 2.项目控件
- 若要添加项目控件，直接在当前目录下添加文件（或者在已有文件中添加代码），代码文件使用 snake_case 命名。

## 三.具体规范
### 1.编码风格
- 在__init__函数中初始化基础控件（除非有动态需求）
- 在_init_ui函数中初始化控件的属性和布局，相关函数也命名为_init_xxx，建议使用snake_case命名法；
- 在_bind_signals函数中绑定信号和槽；
- 所有pyqtSignal的变量名都使用"_s"后缀；
- 使用bg代表背景色，fg代表前景色，bdr代表圆角大小；

### 2.图片资源
- 使用png格式；
- 使用小写命名，不允许带任何字符。推荐使用单个单词命名。
- 添加图片时，亮主题和暗主题图片都需要添加，分别放在ImgPngDay或ImgPngNight文件夹下；
- 添加完图片后，请在UI_design下运行img_to_py.py，生成图片资源文件；
